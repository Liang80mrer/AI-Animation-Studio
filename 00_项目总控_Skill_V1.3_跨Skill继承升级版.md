# AI Animation Studio｜00_项目总控 Skill V1.3｜跨 Skill 继承升级版

# 新增章节｜03A 跨 Skill 强制继承系统

所有 Skill 之间不是自由发散关系。

而是：

上一阶段定义规则
↓
下一阶段必须继承
↓
只能在允许范围内扩展

禁止：
下一阶段推翻上一阶段核心设定。

## 世界观 → 角色

02 世界观必须交付：

角色存在规则
角色来源规则
角色材质规则
角色行为规则
角色职业规则
角色视觉边界
角色禁止方向
角色统一风格规则

03 原创角色必须继承：

角色存在方式
角色材质逻辑
角色行为逻辑
角色视觉语言
角色风格边界
角色禁止方向

禁止：角色脱离世界观。

## 角色 → 故事板

03 原创角色必须交付：

角色身份锁定
角色材质锁定
角色动作习惯
角色表情系统
角色比例规则
角色禁忌项

05 故事板必须继承：

同一个角色
同一个轮廓
同一个材质
同一个表情系统
同一个动作习惯

禁止：故事板阶段重新设计角色。

## 故事板 → Seedance

05 故事板必须交付：

镜头顺序
动作顺序
情绪推进
景别变化
镜头节奏
关键帧建议

07 Seedance 必须继承：

故事板顺序
动作方向
情绪推进
镜头逻辑

禁止：Seedance 随机乱跳镜头。
