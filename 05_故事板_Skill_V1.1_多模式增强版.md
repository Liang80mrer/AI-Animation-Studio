# 01｜故事板模式选择器

在进入故事板之前，你必须先判断：

这个项目应该使用哪一种故事板模式？

不要默认所有故事板都使用同一种逻辑。

不同故事板模式，镜头重点完全不同。

# 02｜故事板模式列表

## A｜动作预演型

适合：

追逐
打斗
变身
丝带舞
清洁分子围攻
机器人拆解
产品机制演示
能量变化
火焰、水流、粒子、布料系统
高节奏视觉桥段

核心主角：

动作系统

动作系统示例：

丝带
水流
毛发
清洁分子
火焰
粒子
布料
能量线
油膜
泡泡
机械臂
光束

每格必须回答：

角色在做什么？
动作系统变成了什么？
空间如何被改变？
镜头如何被卷进去？
这一格比上一格更强在哪里？

核心规则：

角色不是唯一主角。
动作系统负责制造空间变化。
镜头负责进入变化。
每一格都必须推动动作升级。

身体距离怎么变？
停顿在哪里？
情绪有没有被压住？

不要强行动作系统化。

# 04｜动作系统规则（动作预演型专用）

动作系统是：

推动空间变化的视觉系统。

包括：

丝带
水流
毛发
清洁分子
能量线
火焰
粒子
泡泡
布料
油膜
机械结构

规则：

动作系统必须持续变化。
不能静止摆拍。
必须形成：

形态 A
↓
形态 B
↓
形态 C
↓
最终大形态

每格都要：

升级
推进
变化
制造空间改变

# 05｜故事板呈现方式

故事板的目标不是“画漂亮”。

目标是：

动作规划
镜头调度
空间连续性
运动方向
节奏预演

画面风格：

粗略电影分镜
快速草图
手绘铅笔和墨水笔触
速写体块
简化角色
辅助线
动作优先

禁止：

复杂插画
精修概念艺术
重材质渲染
超复杂背景
海报感拼贴

注意：

如果用户明确要“高级故事板美术感”，才允许提高完成度。

——————————————————

# 06｜细节级别规则

默认：

低到中等细节。

重点：

动作可读性
角色识别
镜头关系
空间方向

不是：

材质细节
服装纹理
背景装饰

规则：

动作优先于造型。
环境只保留必要空间线索。

——————————————————

# 07｜节奏与运动逻辑

必须定义：

镜头整体运动节奏。

可选：

快节奏
慢情绪
逐步升级
停顿后爆发
连续推进
断裂式跳切

动作预演型规则：

每格必须有运动。
避免长时间静止。
镜头必须不断被动作卷进去。

情绪表演型规则：

允许停顿。
允许安静。
但情绪必须持续变化。

对话型规则：

信息节奏比动作更重要。
镜头切换必须服务信息变化。

# 10｜场景空间与调度环境

故事板必须发生在：

统一、可连续调度的空间。

不是：

12 张互不相关的漂亮图。

必须解决：

空间连续性
动作尺度
镜头调度
视觉干扰

输出结构：

空间类型：
空间功能：
主要空间元素：
视觉限制：

示例：

空间类型：
微观纤维世界

空间功能：
提供清洁分子与油污战斗空间。

主要空间元素：
纤维森林、水流通道、泡泡层、油膜区。

视觉限制：
背景保持简洁，不抢角色和动作系统。

——————————————————

# 11｜角色在故事板中的使用规则

故事板不是重新设计角色。

而是：

规定角色如何在故事板中表演。

必须写：

角色参考图用途
角色一致性要求
角色动作语言
角色情绪轨迹
角色行为边界
角色与动作系统关系
远景 / 剪影识别规则
多角色关系
连续表演要求

核心规则：

角色像同一个演员连续表演。
不要每格都像重新摆拍。

——————————————————

# 12｜整体镜头语言与拍摄倾向

必须定义：

整体拍摄气质
镜头运动偏好
构图倾向
镜头如何被动作系统带动
镜头升级规则
避免的拍法
最终整体感

示例：

整体气质：
高级动画电影预演板。

镜头偏好：
推进、环绕、俯视旋转、低角度上升。

构图倾向：
强轮廓、纵深层次、前景遮挡。

避免：
连续平拍、角色摆拍感。

# 18｜默认执行逻辑（升级版）

第一步：
判断项目属于哪种故事板模式。

第二步：
定义核心驱动力。

第三步：
确定画幅、分镜数量、布局。

第四步：
确定空间与调度环境。

第五步：
确定角色在故事板中的表演规则。

第六步：
确定整体镜头语言。

第七步：
逐格拆镜头。

第八步：
判断是否需要关键帧扩展。

第九步：
输出完整故事板 Prompt。

第十步：
整理“给下一个 Skill 的输入”，并输出 Seedance 素材建议。
